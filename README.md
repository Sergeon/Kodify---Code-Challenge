##Code Challenge

Code Challenge for [Dokify] (https://dokify.net)


##Instalación

`composer update`  

`php artisan migrate`

`php artisan db:seed`


##Test

`vendor/bin/mat test tests/matura/test_index.php`

Para modificar vistas o scripts y/o utilizar scripts, es necesario ejecutar:
`npm install`
